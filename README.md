# react-native-wallpaper

A simple implementation of a RN library for updating phone wallpaper.

## Installation

```sh
npm install react-native-wallpaper
```

## Usage

```js
import { multiply } from 'react-native-wallpaper';

// ...

const result = await multiply(3, 7);
```
